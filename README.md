## ⚡ Description

**BOWO** adalah advanced security toolkit untuk **Kali Linux** yang menggabungkan tiga fungsi utama dalam satu tools.

Dibuat oleh **VikkDigital069-max** untuk komunitas Kali Linux.

- 🌐 **Network Scanner** (seperti Nmap) - Port scanning, OS detection, service version
- 🔓 **Password Cracker** (seperti John the Ripper) - MD5, SHA1, SHA256 hash cracking
- 🛡️ **ACSL Vulnerability Scanner** - Web vulnerability testing (SQLi, XSS, admin panel finder)
- 🌍 **Web Panel** - HTTP server dengan interface web profesional

## 🔧 Installation

### Requirements
- Kali Linux (or any Debian-based distro)
- Node.js v14+
- Nmap (optional, for better scanning)
- Internet connection

### Step by Step Installation

```bash
# 1. Clone repository
git clone https://github.com/vikkdigital069-max/BOWO.git
cd BOWO

# 2. Install Node.js (if not installed)
sudo apt update
sudo apt install nodejs npm -y

# 3. Install Nmap (optional but recommended)
sudo apt install nmap -y

# 4. Install wordlist (optional)
sudo apt install wordlist -y
sudo gunzip /usr/share/wordlists/rockyou.txt.gz 2>/dev/null

# 5. Make executable
chmod +x bowo.js

# 6. Run as root
sudo node bowo.js
```

🛠️ Troubleshooting

Error: "Command not found: node"

```bash
sudo apt install nodejs -y
```

Error: "nmap: command not found"

```bash
sudo apt install nmap -y
```

Error: "EACCES: permission denied"

```bash
# Run as root
sudo node bowo.js
```

Port scanning slow

```bash
# Reduce port range or use TCP connect scan
Scan type: 2 (TCP Connect)
```
