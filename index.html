⚡ Features

Network Scanner

Port scanning (SYN/TCP/UDP), OS detection, service version seperti Nmap.

Password Cracker

MD5, SHA1, SHA256 hash cracking dengan wordlist rockyou.txt.

ACSL Scanner

Web vulnerability testing: SQLi, XSS, admin panel finder.

Web Panel

HTTP server dengan interface web untuk scanning real-time.

Auto Reporting

Hasil scan otomatis disimpan ke file dengan timestamp.

Lightweight

Dibangun dengan Node.js, ringan dan cepat.

🚀 Installation

Kali Linux / Debian-based

# Clone repository
git clone https://github.com/vikkdigital069-max/BOWO.git
cd BOWO

# Install dependencies
sudo apt update
sudo apt install nodejs nmap -y

# Make executable
chmod +x bowo.js

# Run as root
sudo node bowo.js
